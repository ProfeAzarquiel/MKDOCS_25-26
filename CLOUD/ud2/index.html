    PERFIL DEVOPS

    DevOps es una metodología de desarrollo basada en una serie de prácticas dirigidas a integrar las áreas de Desarrollo y Operaciones, haciendo énfasis en la comunicación, colaboración e integración continua; para asegurar la calidad y automatizar el despliegue de soluciones software.

    Es la metodología en la que están enfocadas la gran mayoría de empresas tecnológicas en la actualidad y por ello, uno de los perfiles más demandado en este campo es el de especialista o ingeniero DevOps. Las habilidades requeridas para dicho puesto, son las siguientes (controlando 4 o 5 de ellas, ya entraremos en las búsquedas de diferentes empresas):

    • Linux principalmente y en bastante menor medida Windows.
    • AWS (Amazon Web Services)
    • Scripting (Bash, Python, PHP).
    • Herramientas de Configuración distribuida (Puppet, Ansible, Chef, Terraform, OpenStack).
    • Administración de servidores web (Apache HTTP, Nginx).
    • Administración y configuración de sistemas de monitoreo (New Relic, Nagios, Grafana, Prometheus, Kibana).
    • Administración y configuración de bases de datos (MySQL, MariaDB, PostgreSQL, MongoDB)
    • Conocimientos de Networking y herramientas (Balanceo, DNS, TCP, UDP, HTTP, telnet, dig, traceroute, iptables)
    • Administración y configuración de sistemas de caché (memcache, redis)
    • Conocimientos de Virtualización y Contenedores (VMWare, Proxmox, Vagrant, Docker, Kubernetes)

    Si englobamos estas habilidades en categorías, nos quedaría algo así:

    • Desarrollo (Habilidades para desarrollar software, haber trabajado como developer)
    • Delivering (conocimientos de herramientas para distribución de deployments)
    • S.O Architect (Arquitectura de S.O, especialmente Linux)
    • Webservers (Administración y configuración de webservers Apache HTTP y/o Ngnix)
    • Troubleshooting (Habilidades para resolución de problemas de desarrollo e infraestructura)
    • Virtualization (Conocimientos y conceptos de virtualización )
    • Cloud (Conocimientos de Cloud, Webservices, APIs, REST, SOAP)
    • Database (Administración y configuración de BBDD relacionales, no relacionales y caché)
